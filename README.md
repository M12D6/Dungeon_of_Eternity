# 🏰 Dungeon of Eternity - Tactical Chaos Edition

A deep roguelike RPG with factions, pets, crafting, housing, achievements, skill trees, subclasses, and 100+ hours of content. Built with React and featuring an innovative elemental enchantment system inspired by Deepwoken.

## ✨ Features

### 🎮 Core Gameplay
- **Deep Roguelike Mechanics**: Procedurally generated dungeons with permadeath
- **Tactical Combat**: Turn-based combat with strategic depth
- **6 Character Classes**: Warrior, Mage, Rogue, Priest, Berserker, Necromancer
- **Skill Trees**: Extensive progression system for each class
- **Hardcore Mode**: Increased difficulty with better rewards

### 🌟 Elemental Enchantment System
- **6 Elemental Types**: Fire, Ice, Lightning, Dark, Holy, Earth
- **Unique Enchantments**: Each element grants special bonuses
  - 🌌 **Astral**: 60% more gold, 40% more exp, 3x consumables (Permanent)
  - 🔥 **Blazing**: Burn status, prevents adaptation (Consumable)
  - ⚡ **Bluster**: 50% explosion damage (Consumable)
  - ❄️ **Frost**: 25% freeze chance (Consumable)
  - 🌑 **Shadow**: Guaranteed crit + 2x damage (Consumable)
  - 🟫 **Earth**: Thorns damage reflection (Consumable)

### 🏆 Advanced Systems
- **Factions & Reputation**: Build relationships with different factions
- **Pet System**: Collect and battle with companions
- **Crafting**: Forge powerful weapons and armor
- **Housing**: Build and upgrade your base
- **Achievements**: 100+ achievements to unlock
- **Boss Battles**: Epic encounters with unique mechanics

## 🚀 Quick Start

### Option 1: Node.js (Recommended)
1. Double-click `start-server.ps1` (Windows) or run `npm start`
2. Server starts automatically and opens the game in 3 seconds

### Option 2: Python
1. Double-click `start-server.bat`
2. Open http://localhost:8000

### Option 3: Visual Launcher
1. Open `launcher.html` in your browser
2. Click "Check if Server is Running"
3. Game opens automatically when server is ready

## 🛠️ Requirements

## 🎯 Game Controls

- **Arrow Keys/WASD**: Move character
- **Mouse**: Navigate menus and click actions
- **Number Keys**: Quick access to abilities
- **ESC**: Return to previous screen
- **F**: Open factions screen
- **H**: Open housing screen
- **T**: Open achievements screen
