🏰 Dungeon of Eternity

Tactical Chaos Edition

📋 How to Start the Game:

Option 1: Node.js (Recommended)
1. Double-click start-server.ps1 (Windows) or run npm start
2. Server will start automatically and open the game

Option 2: Python
1. Double-click start-server.bat
2. Open http://localhost:8000

Option 3: Manual
1. Run python -m http.server 8000 in this folder
2. Open http://localhost:8000
✅ Ready to launch! Choose an option above.
🎮 Open Game (if server is running)
